Introduction
------------
The Conductor WooCommerce Add-On allows WooCommerce Product custom field data to be placed within Conductor Widget output.

Choose from the following custom fields:
* Price
* Add to Cart Button
* Dimensions
* Inventory Stock Quantity
* Inventory Stock Status
* Shipping Weight
* Shipping Length
* Shipping Width
* Shipping Height
* Shipping Class

Troubleshooting
---------------
This add-on requires the following plugins/themes to function properly:
* WooCommerce
* Conductor
* Conductor Custom Fields Add-On
